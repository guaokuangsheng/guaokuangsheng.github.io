## 题目描述

小明正在玩大富翁游戏，他们用的是一种特殊的骰子，骰子的面值只有1和2，此时小明还差N步获得胜利，那么请问小明一共有多少种方式获得胜利呢？

## 输入

一个数字，获得胜利需要的步数

## 输出

输出获得胜利的方式的总和

## 样例输入

```
6
```

## 样例输出

```
13
```

## 提示

N<50



---

* 我的代码

```
#include <iostream>
using namespace std;
int f(int n)
{
if(n==0||n==1)
return 1;
else
return f(n-1)+f(n-2);
}
int main()
{
int n;
cin>>n;
cout<<f(n);
return 0;
}
```
