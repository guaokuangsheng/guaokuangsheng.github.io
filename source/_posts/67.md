- 试题内容  ---   爱消除

  

## 题目描述

老师爱玩消消乐,有一天给班级同学布置了作业,他给出一些数要求按照他的方法消除这些数里重复的数。老师先演示了自己的方法: 如下所示: 

 记一些数有 M[] = x1,x2,x3...

 老师一次修改 x2,x3,...xN.

 当修改ai时候 老师会查看ai 是否在x1 到 xi-1 之中出现过,

 若出现过那么xi 加上 1 如果新的ai还出现过 那么继续给xi 加上 1, 一直到xi没有出现过。

 重复上述过程 一直到修改完最后一个数 那么就消除了这些数中重复的数字了!

 老师给出了最后的答案!

## 输入

第一行包含一个整数 N

第二行包含 N 个整数 : M1 M2 ...



## 输出

依次输出修改完之后的N个整数

## 样例输入

```
4
1 2 1 4
```

## 样例输出

```
1 2 3 4 
```

## 提示




数据范围:




1 <= N <= 100000




1 <= Mi <= 1000000



---

* 我的代码

```
#include <iostream>
#include <cstdio>
using namespace std;
const int N = 1e6 + +1e5 + 5;
 
int fa[N];
bool st[N];
 
int find(int v)
{
    if (!st[v])
    {
        st[v] = true;
        return v;
    }
    return fa[v] = find(fa[v]);
}
 
int main()
{
    for (int i = 0; i < N; ++i) fa[i] = i+1;
    int n;
    scanf("%d", &n);
    for (int i = 0; i < n; ++i)
    {
        int v;
        scanf("%d", &v);
        printf("%d ", find(v));
    }
    return 0;
}
```
