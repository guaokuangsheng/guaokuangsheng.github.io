- 试题内容  ---   连招带师

2. ## 题目描述

   英雄联盟有QWER四个键，不同的释放顺序可以打出不同的连招，假设四个键都按且只按一次那么有多少种连招？请你列举出这些连招。

   

   ## 输入

   无

   ## 输出

   按字典序升序输出所有的连招, 每一个连招的不同按键用空格隔开。每个连招之间使用换行符隔开。

   ## 提示

---

* 我的代码

```
#include <iostream>
#include <algorithm>

int main ()
{

    }
    while ( next_permutation(arr,arr+4) ); 
    return 0;
}
```
