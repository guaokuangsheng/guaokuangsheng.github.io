## 题目描述

Fibonacci 数列中，
F[1] = 1,

F[2] = 1,

对于 i > 3，F[i] = F[i − 1] + F[i − 2]。
尝试求出F[N]/F[N + 1]

## 输入

```
一个正整数 N。(1 ≤ N ≤ 2000000000) 
```

## 输出

```
F[N]/F[N + 1]。答案保留 8 位小数 
```

## 样例输入

```
2
```

## 样例输出

```
0.50000000
```

## 提示

---

* 我的代码

```java
#include<cstring>
#include<iostream>
#include<cstdio>
#include<algorithm>
using namespace std;
typedef long long LL;
const int N = 110;
double cnt[N];
LL n;
 
int main()
{
    cin >> n;
    cnt[1] = 1;
    cnt[2] = 1;
    for(int k = 3; k <= 25; k++)
    {
        cnt[k] = cnt[k-1] + cnt[k-2];
    }
    if(n < 25)
    {
        printf("%0.8f",cnt[n] / cnt[n+1]);
    }else{
        cout << "0.61803399" << endl;
    }
    return 0;
}
```
