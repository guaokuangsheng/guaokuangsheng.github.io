- 试题内容  ---   游玩栖霞山

## 题目描述

学校组织计算机专业的学生去栖霞山游玩，要用校车带同学们去栖霞山。但是校车有限只有一辆所以要租借校外的大巴车带同学们去。假设租借的车和校车的载客量一样，都为50；请问最少需要租借几辆大巴车才能带所有同学去栖霞山（以班级为单位即一个班的学生必须在同一辆车内，一辆车可以载多个班级, 班级的人数不大于50）。

## 输入

输入总共有2行。

第1行是一个正整数n，表示班级的数量。

第2行有n个正整数，分别表示第i个班级的人数为Ci，数字之间使用空格隔开。

## 输出

输出最少需要租借的车辆数。

## 样例输入

```
5
20 30 40 50 10
```

## 样例输出

```
2
```

## 提示


数据约定：


对于100%的数据：


1 ≤ n ≤ 100


1 ≤ Ci ≤ 50





---

* 我的代码

```
#include<stdio.h>
int main()
{
    int n;int a[110]={0};
    scanf("%d",&n);
    for(int i=0;i<n;i++)
        scanf("%d",&a[i]);
    for(int i=0;i<=n-1;i++)
        for(int j=0;j<n-i-1;j++)
            if(a[j]>a[j+1])
            {
                int t=a[j];
                a[j]=a[j+1];
                a[j+1]=t;
            }
    int num=0,sum=50,l=0,r=n-1;
    while(l<=r)
    {
        if(a[l]+a[r]>sum)
        {
            sum=sum-a[r];
            r--;
            num++;
            sum=50;
        }
        else
        {
            sum=sum-a[l]-a[r];
            l++;r--;
        }
    }
    printf("%d",num);
}
```
