- 试题内容  ---  A ? B

题目描述

## 输入

第一行有两个数字A和B，数字之间使用空格隔开。

## 输出

第一行输出A+B的结果
第二行输出A-B的结果
第三行输出A*B的结果
第四行输出A/B的结果
第五行输出A2和B2，两个数之间使用空格隔开。

## 样例输入

```
1 1
```

## 样例输出

```
2.000
0.000
1.000
1.000
1.000 1.000
```

## 提示

数据约定：

对于100%的数据，-109 ≤ A, B ≤ 109 ，且B≠0

注意：

1. 输入的数据不仅仅是整数，方便起见，对于所有的输出，要求保留三位小数。 

2. 因为精度原因，本题请使用**C/C++**做。



---

* 我的代码

```
#include<stdio.h>
int main()
{
    double A,B;
    scanf("%lf%lf",&A,&B);
    printf("%.3lf\n%.3lf\n%.3lf\n%.3lf\n%.3lf %.3lf",A+B,A-B,A*B,A/B,A*A,B*B);
}

```
