- 试题内容  ---   试卷整理

## 题目描述

期中考试结束了。请你帮老师按照分数高低(升序或者降序)整理n份试卷，以便统计同学们考试的排名。分数相同的试卷算作同一个排名。

## 输入

输入一共有三行。

第一行为试卷的数量n。

第二行是排序的顺序seq，seq=1表示升序输出，seq=-1表示降序输出。

第三行共有n个非负整数，其中第i个非负整数表示第i份试卷的分数为xi，每个非负整数之间使用","隔开。

## 输出

输出试卷整理后的结果，分数之间使用空格隔开。

## 样例输入

```
6
1
90,89,87,88,90,88
```

## 样例输出

```
87 88 89 90
```

## 提示

## 数据约定：

1 ≤ n ≤ 1000

0 ≤ xi ≤ 100

seq = -1, 1

注意：

1. 存在分数相同的试卷

2. 注意输出格式

```
#include<stdio.h>
int main()
{
    int n,score,seq;
    int a[101]={0};
    scanf("%d%d",&n,&seq);
    for(int i=0;i<n-1;i++)
    {
        scanf("%d",&score);
        a[score]++;
        getchar(); 
    }
        scanf("%d",&score);
        a[score]++;
    if(seq>0)
    {
    for(int i=0;i<=100;i++)
        if(a[i]!=0) printf("%d ",i);    
    }
    else
    {
    for(int i=100;i>=0;i--)
        if(a[i]!=0) printf("%d ",i);
    }
}
```
