- 试题内容  ---   采药人

## 题目描述

小紫想成为世界上最伟大的医师。今天，她碰到了一个难题。一位有名的医师把她带到一个到处都是草药的山洞里对她说：“这个山洞里有一些不同的草药，采每一株都需要一些时间，每一株也有它自身的价值。如果我给你一段时间，在这段时间里，你可以采到一些草药。我相信你是一个聪明的孩子，应该可以让采到的草药的总价值最大。”

如果你是小紫，你能完成这个任务吗？

## 输入

第一行有两个整数T，M。T表示总共能够用来采药的时间，M表示山洞里的草药数量。
第2~M+1行，每行有两个整数Ti和Wi，Ti表示采摘第i株草药需要的时间，Wi表示第i株草药的价值。

## 输出

输出在规定的时间内可以采到的草药的最大总价值。

## 样例输入

```
70 3
71 100
69 1
1 2
```

## 样例输出

```
3
```

## 提示

数据约定：

对于100%的数据：

1 <= T, Ti <= 1000

1 <= M, Wi <= 100

import java.util.*;
import java.math.*;
import java.io.*;
  
public class Main {
      
    static BufferedWriter out;
    static long ans = 0x3f3f3f3f;
    static long modulo = (long)1e9 + 7;
      
      
    public static void main(String[] args) throws IOException {
        //StreamTokenizer in = new StreamTokenizer(new BufferedReader(new InputStreamReader(new FileInputStream("2020.txt"))));
        out = new BufferedWriter(new OutputStreamWriter(System.out));
        Scanner in = new Scanner(System.in);
        int t = in.nextInt();
        int m = in.nextInt();
        int weight[] = new int[m], value[] = new int[m];
        int dp[] = new int[t + 1];
        for (int i = 0; i < m; i++) {
            weight[i] = in.nextInt();
            value[i] = in.nextInt();
        }
        for (int i = 0; i < m; i++) {
            for (int j = t; j >= 0; j--) {
                if (j >= weight[i]) {
                    dp[j] = Math.max(dp[j], dp[j - weight[i]] + value[i]);
                }
            }
        }
        out.write(String.valueOf(dp[t]));
        out.flush();
    }
}
