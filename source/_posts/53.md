- 试题内容  ---  输出字符画

## 题目描述

使用printf或cout将如下的字符画输出一遍即可完成此题： 
  .--    --.
  ( \---/ )
 ._/ o o \_.
  {= ^ =}
   \> - < [hello big-data]
   /     \
 // 2020 \\
 //|  ----  |\\

## 输入

无

## 输出

原模原样地输出一遍题目中给出的字符画

## 提示

必须和给出的字符画完全一致才能算通过，所以在提交前务必反复检查

---

* 我的代码

```
#include<stdio.h>
 
int main()
{
    printf("   .--       --.\n   (  \\---/  )\n  ._/ o  o \\_.\n   {=  ^  =}\n     >  -  < [hello big-data]\n     /         \\\n  //  2020  \\\\\n //|   ----    |\\\\\n");
    return 0;
}
```
